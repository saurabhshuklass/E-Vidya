# E-Vidya 📚

An Android-based E-Learning Quiz Application developed using Kotlin and Firebase.

## Features

- User Registration & Login
- Firebase Authentication
- Forgot Password
- Profile Management
- Quiz History
- Random Quiz Questions
- Progress Tracking
- Timer Based Quiz
- Result Analytics
- Multiple Subjects
- Multiple Standards (9th–12th)
